The Control class is responsible for reading the input file (function construct_input() called in the Control class constructor) and evaluating the signals and control blocks*. All input data are stored in the dictionary reactor.inp. If the control block requires integration of an Ordinary Differential Equation (ODE) with respect to time, the right-hand side of this ODE is included* in the overall system of ODEs through the calculate_rhs(self, reactor, t) function called by the Reactor class constructor.

*under development
